Buttons

Simple CSS for buttons.

Default Buttons

To create a Pure Button, add the btn classname to any <a> or <button> element.

A Pure Button 
<a class="btn" href="#">A Pure Button</a>
<button class="btn">A Pure Button</button>

Disabled Buttons

To mark a button as disabled, add the btn-disabled classname alongside btn. Alternatively, add the "disabled" attribute directly to your button. 

<button class="btn btn-disabled">A Disabled Button</button>
<button class="btn" disabled>A Disabled Button</button>

Active Buttons

To style a button so that it appears "pressed", add the btn-active classname alongside btn.

An Active Button 
<a class="btn btn-active" href="#">An Active Button</a>
<button class="btn btn-active">An Active Button</button>

Primary Buttons

To indicate that a button represents a primary action, add the btn-primary classname alongside btn.

A Primary Button 
<a class="btn btn-primary" href="#">A Primary Button</a>
<button class="btn btn-primary">A Primary Button</button>

Customizing Buttons

Thanks to Pure's minimal styling, it is easy to build off of the generic Pure button and create customized buttons for your own application.

To customize button styles, you should group your custom CSS into a class such as button-foo, which you can then add to an element that already has the btn classname. Here are some examples.

Colored buttons with rounded corners

<div>
    <style scoped>

        .button-success,
        .button-error,
        .button-warning,
        .button-secondary {
            color: white;
            border-radius: 4px;
            text-shadow: 0 1px 1px rgba(0, 0, 0, 0.2);
        }

        .button-success {
            background: rgb(28, 184, 65); /* this is a green */
        }

        .button-error {
            background: rgb(202, 60, 60); /* this is a maroon */
        }

        .button-warning {
            background: rgb(223, 117, 20); /* this is an orange */
        }

        .button-secondary {
            background: rgb(66, 184, 221); /* this is a light blue */
        }

    </style>

    <button class="button-success btn">Success Button</button>
    <button class="button-error btn">Error Button</button>
    <button class="button-warning btn">Warning Button</button>
    <button class="button-secondary btn">Secondary Button</button>
</div>

Buttons with different sizes

<div>
    <style scoped>

        .button-xsmall {
            font-size: 70%;
        }

        .button-small {
            font-size: 85%;
        }

        .button-large {
            font-size: 110%;
        }

        .button-xlarge {
            font-size: 125%;
        }

    </style>

    <button class="button-xsmall btn">Extra Small Button</button>
    <button class="button-small btn">Small Button</button>
    <button class="btn">Regular Button</button>
    <button class="button-large btn">Large Button</button>
    <button class="button-xlarge btn">Extra Large Button</button>
</div>

Buttons with icons

Pure doesn't ship with icon fonts, but we play nice with existing ones. Incorporating icon fonts with Pure Buttons is easy. In the example below, we're using icon fonts from Font Awesome. Put the Font Awesome CSS file on your page and use an <i> element within a btn element.

Checkout 
<button class="btn">
    <i class="fa fa-cog"></i>
    Settings
</button>

<a class="btn" href="#">
    <i class="fa fa-shopping-cart fa-lg"></i>
    Checkout
</a>
